Trip Planner Website : TravelBoxTrip
Technologies Employed
  • Angular Framework
  • Google Maps API
  • HTML
  • CSS
Developed a web application that integrates a map-based feature to assist travelers in identifying suitable attractions during their journeys. The application also enables the customization of trip plans based on the preferences of tourists, ensuring a comprehensive and tailored travel experience. The map displays the distances between attractions and provides intelligent suggestions for inclusion, enhancing the overall planning and optimisation of the trip.
