Custom Academic Portfolio: An External Project
Technologies Employed
  • React Framework
  • TypeScript
  • Github pages
This custom-designed portfolio project is meticulously crafted to be published on GitHub Pages as a singular-page application. The project's primary objective is to serve as an exhaustive platform for academic degree holders to present their research and project-related data. It will function as a centralised hub to display all the qualifications and personal accomplishments of the degree holder, thereby rendering it an appealing resource for academic recruiters. The project is structured into tabs and includes a downloadable feature for a curriculum vitae.
